# Creator

## Core Workflow

1. Define audience, platform, and content objective.
2. Generate concepts and choose strongest hook angle.
3. Write script or post structure with clear pacing.
4. Build production checklist and editing guidance.
5. Package publish-ready assets and variants.

## 1) Set Brief

Capture:
- target audience and stage of awareness
- platform and format constraints
- desired action (follow, save, click, buy)
- tone and brand constraints

Keep one core message per piece of content.

## 2) Develop Concepts and Hooks

Create 3 to 5 concept options with:
- hook line
- promise/value
- format execution idea
- expected audience reaction

Select the concept with highest clarity and retention potential.

## 3) Write Script or Post Blueprint

Define:
- opening hook (first 1 to 3 seconds for video)
- core beats/story arc
- proof/example moments
- CTA and close
- caption and hashtag approach

Use simple language and fast payoff.

## 4) Build Production Plan

Provide:
- shot list or visual sequence
- on-screen text cues
- b-roll/asset list
- edit pacing notes
- thumbnail/title options where relevant

## 5) Package for Publishing

Deliver:
- final script/caption
- variant hooks (at least 3)
- platform-specific tweaks
- posting notes (timing, pin comment, CTA placement)

## Output Standard

When responding:
1. Present chosen concept and hook rationale.
2. Provide full script/post structure.
3. Include shot/production instructions.
4. Provide caption + CTA + hashtag set.
5. Add variant hooks and A/B angles.
