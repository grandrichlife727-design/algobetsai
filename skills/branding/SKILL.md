# Branding

## Core Workflow

1. Define brand objective, audience, and market context.
2. Establish positioning and core narrative.
3. Build messaging architecture and voice.
4. Define visual identity foundations.
5. Set brand governance and rollout plan.

## 1) Define Brand Context

Capture:
- category and competitive set
- ideal audience and use context
- current perception problems
- desired perception shift

Anchor decisions to business goals, not aesthetics alone.

## 2) Establish Positioning

Define:
- target audience definition
- frame of reference
- point of difference
- reason to believe
- brand promise

Keep positioning clear enough to guide product and marketing choices.

## 3) Build Narrative and Messaging

Create:
- brand story (problem, change, outcome)
- messaging hierarchy (master message, pillars, proof points)
- elevator pitch and one-liner variants
- objection handling statements

Ensure each message reflects the positioning without jargon.

## 4) Define Verbal and Visual Foundations

Set verbal system:
- tone attributes
- voice dos/don'ts
- writing examples by channel

Set visual system:
- color intent and palette roles
- typography roles
- imagery style
- icon/shape principles
- logo usage principles (if available)

## 5) Set Governance and Rollout

Define:
- brand guideline structure
- approval workflow
- asset ownership
- adoption milestones by team/channel

Plan for evolution with versioned updates.

## Output Standard

When responding:
1. State positioning in one concise paragraph.
2. Provide messaging pillars and proof.
3. Define tone and voice rules.
4. Outline visual principles and constraints.
5. End with rollout and governance steps.
